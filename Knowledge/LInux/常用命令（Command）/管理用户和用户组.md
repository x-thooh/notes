## 用户
### 查询用户

# 当前用户,登录的用户
whoami
# 用户信息
id

### 添加用户
useradd [options] username
# 用户添加，指定家目录
useradd username -d homedir

### 用户信息文件
- /etc/passwd：用户帐号信息文件。
    用户名:口令：用户标识号：用户组标识号：注释性描述：主目录：登录shell
- /etc/shadow：用户帐号信息加密文件。
    用户名：密码：最后一次修改时间：最小时间间隔：最大时间间隔：警告时间：不活动时间：失效时间：标志 密码长度应至少为 8-12 个字符，包括特殊字符、数字、小写字母等。 通常密码格式设置为$id$salt$hashed $id是在 GNU/Linux 上使用的算法如下： $1$ is MD5 $2a$ is Blowfish $2y$ is Blowfish $5$ is SHA-256 $6$ is SHA-512 # MD5,SHA-256,SHA-512 是crypt的方法，不是标准的方法
- /etc/group：组信息文件。
    用户组标识号：组内用户列表
- /etc/default/useradd：定义默认设置文件。
- /etc/login.defs：系统广义设置文件。
- /etc/skel：默认的初始配置文件目录。

### 修改密码
password [username]

### 修改用户shell设置
usermod -s shell_path username
usermod -s /bin/bash username

### 修改主目录
usermod -d home_dir username

### 修改UID
usermod -u UID username

### 修改账号的有效期
usermod -e MM/DD/YY username

### 删除用户
userdel username
# 强制删除
userdel -f username
# 用户删除，以及家目录
userdel -r username

### 用户授权
[[sudo]]

### 切换用户
# 切换用户
su - root
# 退出
exit
# 注销用户
logout

### su & su - & sudo 区别
# 切换用户, su 等效 su root
su username
# 切换用户，同时变更环境变量，工作目录等相关的信息， su - 等效 su - root
su - username
# 授权的su，需要提供当前用户的密码
sodo command

## 用户组
### 添加用户组
groupadd [options] groupname

### 用户组信息
- /etc/gshadow：用户组信息加密文件。
- /etc/group：组信息文件。
- /etc/login.defs：系统广义设置文件。

### 修改用户组
### 修改GID
groupmod -g GID groupname

### 修改用户组名
groupmod -n newgroupname oldgroupname

### 删除用户组
groupdel groupname

### 将用户加入用户组或从用户组移除
gpasswd -a user groupname

### 切换用户组
一个用户同时属于多个用户组时，在该用户登录后，使用newgrp命令可以切换到其他用户组，以便具有其他用户组的权限
newgrp groupname
