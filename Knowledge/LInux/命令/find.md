# find

## 摘要

```
find [搜索路径] [匹配条件] [执行动作]
```



## 搜索路径
> [!tip] 如果不指定路径，默认在当前目录下搜索。

支持绝对路径与相对路径

## 匹配条件
> [!tip] 条件之间默认是“与”（AND）的关系。

### 文件名称

