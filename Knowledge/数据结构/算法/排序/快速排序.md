# 快速排序（Quck Sort）
## 概念
通过一趟排序将待排序记录分割成独立的两部分，其中一部分记录的关键字均比另一部分记录的关键字小，则可分别对这两部分记录继续进行排序，以达到整个序列有序的目的。
![](https://raw.githubusercontent.com/x-thooh/picture/main/obsidian/picture849589-20171015230936371-1413523412.gif)

## 代码

```go
func Quick(list []uint32) []uint32 {
	if len(list) <= 1 {
		return list
	}
	left, right := 0, len(list)-1
	identify := list[left]
LOOP:
	for left < right {
		for list[right] >= identify {
			right--
			if left == right {
				break LOOP
			}
		}
		list[left] = list[right]
		left++
		if left == right {
			continue
		}
		for list[left] <= identify {
			left++
			if left == right {
				break LOOP
			}
		}
		list[right] = list[left]
		right--
		if left == right {
			continue
		}
	}
	list[left] = identify
	Quick(list[:left])
	Quick(list[left+1:])
	return list
}
```

> [!tip] O(n) 时间复杂度内求无序数组中的第 K 大元素
