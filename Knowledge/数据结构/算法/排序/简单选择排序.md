# 简单选择排序
## 概念
通过`n-i`次关键字间的比较，从`n-1-i`个记录中选出关键字最小的记录，并和第`i（i≤i≤n）`个记录交换之。
![](https://raw.githubusercontent.com/x-thooh/picture/main/obsidian/picture849589-20171015224719590-1433219824.gif)

## 代码

```go
func choose(list []uint32) []uint32 {
	for i := 0; i < len(list); i++ {
		chIndex := i
		for j := i + 1; j < len(list); j++ {
			if list[chIndex] > list[j] {
				chIndex = j
			}
		}
		if chIndex != i {
			list[i], list[chIndex] = list[chIndex], list[i]
		}
	}
	return list
}
```
