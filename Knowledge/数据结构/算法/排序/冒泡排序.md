# 冒泡排序
## 概念

两两比较**相邻记录**的关键字，如果反序则交换，直到没有反序的记录为止。
错误思路:

让每一个关键字，都和它后面的每一个关键字比较，如果大则交换，这样第一位置的关键字在一次循环后一定变成最小值。

## 代码

```go
func bubble(arr []int) {
	n := len(arr)
	for i := 0; i < n-1; i++ {
		// 每次迭代将最大的元素冒泡到最后
		for j := 0; j < n-i-1; j++ {
			// 如果当前元素比下一个元素大，则交换它们
			if arr[j] > arr[j+1] {
				arr[j], arr[j+1] = arr[j+1], arr[j]
			}
		}
	}
}
```

```go
func bubble(list []uint32) []uint32 {
	var (
		tempPosition = 0
		length       = len(list) - 1
		tempLength   = length
	)
	for i := 0; i < length; i++ {
		flag := false
		for j := 0; j < tempLength; j++ {
			if list[j] < list[j+1] {
				list[j], list[j+1] = list[j+1], list[j]
				tempPosition = j
				flag = true
			}
		}
		tempLength = tempPosition
		if !flag {
			break
		}
	}
	return list
}
```
